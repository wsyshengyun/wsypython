## 卸载软件

```
sudo apt --purge remove 软件名　　#卸载掉配置文件　
sudo apt remove 软件名
sudo dpkg -r 软件名
```

## 安装软件

**deb包**

```
sudo kpkg -i  bao.deb 
```

## 浏览安装的文件

```
dpkg --list
```
